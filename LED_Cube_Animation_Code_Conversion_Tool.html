Convert the output numbers of the LED cube animation in Unity3D to the code available for Arduino.

Designed by Wan-Ting Hsieh (2016)
Paste the output number from Unity3D into the right column.: ex. 0,22,33,99,155,266,577,688,911：

STEP 1: Copy the generated code and paste it into Arduino IDE.

STEP 2: Add the following gray parts manually：

void function "name of the animation" + "number of the frame"{ the generated code };


ex.
void starFirstImg(){

digitalWrite(layer[0],1); //Anode Layer

digitalWrite(column[11],0); //Cathode Columns
digitalWrite(column[22],0);
digitalWrite(column[33],0);

delay(1);
turnEverythingOff();
};
